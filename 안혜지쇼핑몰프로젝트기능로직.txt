1.  사용자 인증 (회원가입, 로그인, 로그아웃)
사용자 정보를 등록하기 위한 회원가입 API
사용자 인증과 권한 관리를 위한 로그인 및 로그아웃 API


 

2. 상품 관련 (상품 목록 조회, 상품 상세 조회)
 
상품 목록을 가져오기 위한 상품 목록 조회 API
 
특정 상품의 상세 정보를 가져오기 위한 상품 상세 조회 API

 


(장바구니기능은 추후에 넣어도 상관없을듯)
3. 장바구니 관련 (장바구니 추가, 조회, 상품 수량 변경, 삭제)
 
상품을 장바구니에 추가하기 위한 장바구니 추가 API
 
현재 장바구니에 담긴 상품 목록을 조회하기 위한 장바구니 조회 API

4. 장바구니 내 상품 수량을 변경하기 위한 상품 수량 변경 API
 
장바구니에서 상품을 삭제하기 위한 상품 삭제 API

 


5. 주문 관련 (주문 생성, 주문 조회, 주문 상태 변경)
 
사용자의 주문을 처리하기 위한 주문 생성 API
 
주문 내역을 조회하기 위한 주문 조회 API
 
주문 상태를 변경(예: 배송 중, 배송 완료)하기 위한 주문 상태 변경 API
